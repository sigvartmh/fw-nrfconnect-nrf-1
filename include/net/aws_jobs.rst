AWS Jobs
########

This provides functions for working with the `AWS IoT jobs
<https://docs.aws.amazon.com/iot/latest/developerguide/iot-jobs.html>`_
service.

Configuration
*************

Use Kconfig to configure the MQTT message buffer size.
