.. _iso_test:

Isochronous channels and ACL tests
##################################

.. contents::
   :local:
   :depth: 2

These tests are made to verify BIS and ACL simultaneous data transmission.
Compared to other tests, these run as integration tests involving both the Controller, Host and application layer.


Requirements
************

The tests support the nrf5340_audio_dk development kit.

Note
****
These tests are run as part of Nordic CI with specific configurations.
It should not be treated as a reference for general use, as it is designed for CI and fault injection purposes.
